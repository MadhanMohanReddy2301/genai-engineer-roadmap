## 🚀 How to Use

<img align="right" src="https://cdn-icons-png.flaticon.com/256/8637/8637101.png" width="180" />

1. 🎯 Start from **Phase 1** and move in order
2. 🧭 Don't skip the **phase gates** — explain each topic in your own words
3. 🛠️ Build a **small project** after every two phases
4. 📈 Track **quality, latency, and cost** for every project
5. 🔄 Revisit **monthly** to refine prompts, retrieval, and evals

<br clear="right"/>

## 🖥️ Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/MadhanMohanReddy2301/genai-engineer-roadmap.git

# 2. Move into the folder
cd genai-engineer-roadmap

# 3. Open in your browser
start index.html    # Windows
open  index.html    # macOS
xdg-open index.html # Linux
```

> 💡 **Tip:** No build tools, no dependencies — it's just a single HTML file.

## 🌐 Deploy to GitHub Pages

<div align="center">

![Deploy](https://img.shields.io/badge/Deploy-GitHub_Pages-222?style=for-the-badge&logo=githubpages&logoColor=white)
![Free](https://img.shields.io/badge/Cost-Free-success?style=for-the-badge)
![Time](https://img.shields.io/badge/Setup_Time-2_min-blue?style=for-the-badge)

</div>

1. Push this repo to GitHub
2. Ensure your file is named **`index.html`**
3. Go to **Settings → Pages**
4. Select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Save → Your live link:
   ```
   https://<your-username>.github.io/<repo-name>/
   ```
